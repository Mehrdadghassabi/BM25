# BM25

1- clone this repository
```
    git clone https://github.com/Mehrdadghassabi/BM25.git
```

2- change directory to BM25
```
    cd BM25/
```

3- install hazm
```
    pip3 install hazm
```

4- run the project
```
    python3 main.py
```
